文字教程


  1. 下载官网的游戏键盘配置文件和游戏本体
  2. 安装后点击"游戏键盘",按应用内的教程开启"游戏键盘"输入法
  3. 进入游戏键盘设置,导入"DELTARUNE.gkp",并勾选"左右横滑切换键盘"和"热键→显示/隐藏→音量+"
  4. 进入游戏,按下"音量+"按键,调出手柄
  5. 选择一个空存档,左右横滑,从手柄换为键盘,打开大写锁定,点击"Z"键两次
  6. 进入游戏,点击手柄图标返回手柄,开始游玩 ( A 键确认, B 键取消, X 键菜单 )
